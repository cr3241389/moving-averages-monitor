# Bitcoin Moving Averages Monitor

A real-time visualization tool for monitoring Bitcoin and other cryptocurrency moving averages.

## Features

- Real-time EMA (Exponential Moving Average) and SMA (Simple Moving Average) indicators
- Support for multiple cryptocurrencies (BTC, ETH, BNB, SOL)
- Multiple timeframes (1m, 5m, 15m, 30m, 1h, 4h, 1d)
- Visual density indicator showing when moving averages are aligned
- Interactive chart with detailed information panel

## Indicators

- EMA 20, EMA 60, EMA 120
- SMA 20, SMA 60, SMA 120

## Usage

Simply open `ma_monitor.html` in any modern browser to start monitoring moving averages.

## Technical Details

- Uses Binance API for real-time market data
- Built with Chart.js for visualization
- Responsive design for desktop and mobile
